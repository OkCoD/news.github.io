NoordveenRP Keybinds



-=Bewegen=-
W = Vooruit lopen
A = Naar links lopen
D = Naar rechts lopen
S = Naar achteren lopen
Q = Schuilen
E = Interactie
R = Slaan
Y = Armen over elkaar
X = Handen omhoog
Ctrl = Hurken
Spatie = Springen
/me heeft papieren = Interacties: geef / zie rijbewijs of ID kaart
~ = Aanpassen voice range

-=Voertuig Interactie=-
F = Instappen
L = Voertuig op slot
Delete = Voertuig Aan/Uit
Backspace = Gevaren lichten
- = Kofferbak open doen / inhoud bekijken (Moet je voor achter de auto staan)

-=Wapen Interactie=-
R= Herladen
Tab = Wapen wegstoppen/uithalen
Rechter muis knop = Mikken
Linker muis knop = Schieten

-=Camera=-
C = Achteruit kijken
V = Camerapunt veranderen

-=Overige=-
F2 = Inventaris
F3 = Persoonlijk Menu
F3 = Animaties
T = Chatbox (ooc & /twt
M = Mobiel
K = accessoires (Hoeden, Petten,masker etc)
Z = Overheidsdiensten online
N = Praten (Push to talk)
P/Esc = Kaart

-=Commando's=- /me = Actie ondernemen
/do = Actie ondernemen
/twt = in twitter praten
/repareer = Repareer je auto als er geen ANWB aanwezig is
/baan = Laat je baan zien
/bank = Laat je bank geld zien (dit valt ook te vinden op je telefoon)
/cash = Laat je cash geld zien
/showsociety = Laat je bedrijfsrekening geld zien
/showdirty = Laat je zwart geld zien
